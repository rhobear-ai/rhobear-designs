# Swarm briefs — RHOBEAR Designs

Every worker brief Iron Man dispatches for this project lives here, version-controlled.

**Iron Man canon:** whenever a plan is made, all of its briefs are written into this folder and
pushed to git — so the plan is always available, auditable, and re-runnable. Briefs are the exact,
self-contained specs handed to swarm workers (MiniMax-M3 / high) on chat-brain-1.

Naming: `<lane-id>.md` (e.g. `td1-3d-core.md`). Each brief is self-contained: scope, guardrails,
task, Definition of Done, and the commit/push/open-PR finish.

# Rhobear Designs

Design assets and resources for the RHOBEAR brand.

## Structure

- `brand/` — Logo, colors, typography guidelines
- `templates/` — Reusable design templates
- `assets/` — Icons, illustrations, imagery

## Brand Colors

- Primary: `#1A1A2E` (Deep Navy)
- Accent: `#E94560` (Rhobear Red)
- Neutral: `#F5F5F5` (Off-White)

What it is

A real editor on top of your own files, not a walled garden.

Most builders trap your work behind their renderer. RHOBEAR Designs opens the HTML you already have, edits the actual DOM, and writes it back out clean. You can leave whenever you want and take everything with you.

Canvas

Click, drag, resize — on the page itself

Select any element and the inspector follows. Drag blocks in, resize with handles, double-click to type. What you move is the document, not a preview of it.

Import

Open a file or a folder

A single .html or a whole project with linked images and CSS.

Export

Clean code, every time

Download the document with inlined styles, or a ZIP of index.html + styles.css + assets.

Layers

See the structure

The element tree on one side, the style controls on the other. No mystery markup.

Devices

Desktop, tablet, phone

Switch widths to check the layout before you ship it.

AI, as a chat bubble

Ask for the change. Watch it land without the flicker.

Connect any model with your own key. Type what you want in plain language and the editor applies it as a clean edit to the canvas, not a full reload. The AI never holds your project hostage; it just helps you move faster.

How it works

Four steps. None of them lock you in.

01

Open

Point it at an HTML file or a project folder from your disk.

02

Edit

Move, restyle, and rewrite on the live canvas. Undo and redo the whole way.

03

Ask

Hand the tedious changes to your model through the chat bubble.

04

Export

Save the HTML or a full ZIP. Host it anywhere. You own it.

Open source

MIT licensed, runs on your machine, reads its own source out loud.

The editor is GrapesJS for the canvas and serializer, wrapped in RHOBEAR chrome, with original code for import, export, and the AI bubble. Clone it, fork it, sell what you build with it. The Playwright suite checks that every toolbar button does what it says.
